# Credits & Attribution

## Imagery
This site currently uses **no external or copyrighted images**. All visuals are generated in-browser
with CSS gradients, an HTML5 Canvas starfield, and inline SVG. This avoids licensing issues and keeps
the site fast and self-contained.

### Optional NASA/ESA/Chandra imagery (not yet included)
If high-resolution astronomy imagery is added later (e.g. into `assets/img/astro/`), it should be
**public domain or properly licensed**, with attribution recorded here. Suggested public-domain sources:

- NASA Image and Video Library — https://images.nasa.gov (most NASA imagery is public domain)
- Chandra X-ray Observatory image gallery — https://chandra.harvard.edu/photo/
  (credit line typically "NASA/CXC/…"; check the per-image credit)
- ESA/Hubble — https://esahubble.org/images/ (CC BY 4.0 unless noted)

**Attribution template (fill in when an image is added):**
`assets/img/astro/<file> — Credit: NASA/CXC/<institution>; Source: <URL>; License: <PD or CC BY 4.0>`

> Note: No imagery on this site implies that the data, observations, or results shown are personally
> produced by Karthik Balasubramaniam. Scientific results referenced are described in the cited
> peer-reviewed publications.
